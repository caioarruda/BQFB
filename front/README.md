# FB-Quest FrontEnd

Projeto com objetivo de demonstrar a capacidade técnica do desenvolvedor.
link: http://prontdoc.com.br:3000/

## Instalação

Use o NPM para gestão de pacotes

```bash

cd front
npm install

```

## Uso

Inicializar o servidor de desenvolvimento.

```bash

cd front
npm start

```
